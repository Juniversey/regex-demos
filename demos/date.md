#### 匹配yyyy-mm-dd 格式日期
```javascript
  2017-06-10
```
分析:
年，四位数字即可，可用 [0-9]{4}。
月，共 12 个月，分两种情况 "01"、"02"、...、"09" 和 "10"、"11"、"12"，可用 (0[1-9]|1[0-2])。 日，最大 31 天，可用 (0[1-9]|[12][0-9]|3[01])。
```javascript
var regex = /^[0-9]{4}-(0[1-9]|1[0-2])-(0[1-9]|[12][0-9]|3[01])$/; 
console.log( regex.test("2017-06-10") );
// => true
```
